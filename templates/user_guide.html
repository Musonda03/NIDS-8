User Guide

Introduction

This Network Intrusion Detection System (NIDS) monitors network traffic and detects potential threats using machine learning techniques.

Features

  • Real-time traffic visualization
  • Detailed anomaly detection
  • User-friendly dashboard

How to Use

1. Login to the system using your credentials.

2. Navigate to the dashboard to monitor real-time network traffic and detect anomalies.

3. Use the 'View Traffic Visualization' and 'View Plotly Chart' buttons to view detailed charts.

Interpreting Results

Anomalies detected by the system are highlighted on the dashboard. Click on each anomaly card for detailed information.
